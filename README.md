# Détreakt

Playable grid of [Détrak](https://www.gigamic.com/jeux-de-voyage/511-detrak-3421271117919.html), with placement
indications and auto-scoring.

![game.gif](doc%2Fgame.gif)

## How to

`npm install`

`npm run dev`

`npm run build`
